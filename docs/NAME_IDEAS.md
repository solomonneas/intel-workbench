# Alternative Repo Names

Working name: `intel-workbench`

## 1. `evidence-atlas`
Display name: Evidence Atlas

Best if the project should feel like a polished product. It covers ACH, Diamond Model, IOC extraction, source mapping, and future analytic workflows without boxing the repo into one method.

## 2. `analysis-cell`
Display name: Analysis Cell

Best if the repo should feel operational and serious. Short, clean, and closer to a cyber intel shop than a generic dashboard.

## 3. `hypothesis-forge`
Display name: Hypothesis Forge

Best if you want something more memorable. It directly points at competing hypotheses and the process of shaping evidence into an assessment.

## Recommendation

Pick `evidence-atlas` if this becomes a portfolio product. Pick `analysis-cell` if you want a sharper SOC/intel-ops feel. Keep `intel-workbench` if search clarity matters more than branding.
