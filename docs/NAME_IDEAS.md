# Alternative Repo Name Ideas

Working name: `intel-workbench`

## Strongest options

1. `analysis-cell`
   - Display name: Analysis Cell
   - Why: Short, operational, and broader than ACH without sounding generic.

2. `evidence-atlas`
   - Display name: Evidence Atlas
   - Why: Fits ACH, Diamond Model, IOC extraction, and future source-mapping features.

3. `hypothesis-forge`
   - Display name: Hypothesis Forge
   - Why: Memorable, action-oriented, and directly tied to competing hypothesis work.

4. `tradecraft-studio`
   - Display name: Tradecraft Studio
   - Why: Professional and expandable if the tool grows beyond cyber threat intel.

5. `dossierdesk`
   - Display name: DossierDesk
   - Why: Brandable and compact, with a clear analyst workspace feel.
